# Hello World – Java Wordle Clone 🎮

A simple Wordle-style game written in Java for CS 221 proj

 Project Files

- `Main.java` – Starts and runs the game  
- `Wordle.java` – Handles game logic  
- `Backend.java` – User interaction and validation  
- `Words.java` – Loads word list  
- `5-words.txt` – List of valid 5-letter words used for the game  

How to Run This Java Project

Prerequisites
- Java JDK 8 or higher
- Terminal or Command Prompt

Steps

```bash
git clone https://github.com/doa5302/hello-world.git
cd hello-world
javac *.java
java Main
